Android Chrome Download Origin Spoof

CWE-451 Data URI Abuse Android Bypass Chrome 143+

Attack Vector

Crafting a data: URI with an embedded hostname tricks Chrome's download manager into displaying a spoofed origin. On Android, combining this with fullscreen mode hides the real URL bar, making the attack significantly more convincing. 

data:https://google.com/[...250 spaces...]application/vnd.android.package-archive;base64,[PAYLOAD]

Bypass #1: Fullscreen + Download (Most Effective on Android)

Fullscreen hides the URL bar. Victim only sees the spoofed download origin.

Bypass #2: Scroll-to-Hide URL Bar (Android)

On Android Chrome, scrolling down hides the URL bar naturally. Trigger download after scroll.

Bypass #3: Touch Event Chaining

Chain touch events to bypass user-gesture requirements and trigger download.

Bypass #4: PWA / Standalone Mode

If the site is added to home screen (PWA), there is NO URL bar at all.

Bypass #5: Intent Scheme + Download Combo

On Android, intent:// URLs can open apps. Combined with download spoof.

Bypass #6: Convincing File Types

Different file extensions that appear more legitimate to victims.
